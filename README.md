# Portfólio Profissional – Guilherme Mendes Araújo

Este repositório contém o código-fonte do meu **site de portfólio profissional**, desenvolvido para apresentar minha atuação em **liderança de operações de TI, infraestrutura crítica e ambientes distribuídos**.

O objetivo do projeto não é demonstrar apenas habilidades técnicas, mas **contexto real de decisão, responsabilidade assumida e impacto operacional**, com foco em recrutadores técnicos e gestores de tecnologia.

## Objetivo do Portfólio

- Apresentar trajetória profissional de forma clara e objetiva
- Demonstrar experiência prática em operações de TI e liderança técnica
- Evidenciar decisões, riscos e resultados em projetos reais
- Servir como material complementar ao currículo em processos seletivos

## Estrutura do Conteúdo

O site é dividido em duas camadas principais:

- **Institucional**: visão geral, perfil profissional e posicionamento
- **Técnica**: cases detalhados com contexto, decisão-chave, execução e resultados

Essa separação garante leitura rápida para gestores e aprofundamento técnico para quem desejar.

## Tecnologias Utilizadas

- React
- TypeScript
- Vite
- Estrutura baseada em componentes reutilizáveis
- Conteúdo desacoplado do código (JSON)


## Observação

Este repositório **não representa um produto comercial ou serviço**, mas sim um portfólio profissional com foco em **empregabilidade, liderança e maturidade técnica**.


# Professional Portfolio – Guilherme Mendes Araújo

This repository contains the source code of my **professional portfolio website**, designed to present my experience in **IT operations leadership, critical infrastructure, and distributed environments**.

The goal of this project is not only to showcase technical skills, but to highlight **real-world decision-making, ownership, and operational impact**, targeting technical recruiters and technology managers.

## Portfolio Purpose

- Present professional background in a clear and objective way
- Demonstrate hands-on experience in IT operations and technical leadership
- Highlight decisions, risks, and outcomes from real projects
- Serve as a complementary asset to my résumé during hiring processes

## Content Structure

The website is organized into two main layers:

- **Institutional**: professional positioning and high-level profile
- **Technical**: detailed case studies including context, key decisions, execution, and results

This structure allows for fast evaluation by managers and deeper technical exploration when needed.

## Technologies Used

- React
- TypeScript
- Vite
- Reusable component-based architecture
- Content decoupled from code (JSON-based)

## Note

This repository **does not represent a commercial product or service**, but a professional portfolio focused on **employability, leadership, and technical maturity**.
